form-input

form : input태그의 값을 전달하는 전체 태그

input : 값을 입력받는 태그

input 태그를 이용해서 값을 전달 할 때
name과 value값을 통해서 전달

input 타입에 submit를 넣으면 value에는 버튼에 들어갈 문장 입력이 되고

input 타입에 button을 만들어 눌려도 당장은 반응이 없다. 단순 클릭할 수 있는 공간을 만들어준다

대신 button 태그에 타입을 submit으로 하면 input-submit랑 같다
